##  Project Overview

The **Bright Future University Website** is a modern educational website that includes multiple pages such as Home, About, Courses, Admissions, Campus Life, and Contact.

The goal of this project was to:

* Practice responsive web design
* Implement clean UI/UX principles
* Structure a multi-page website
* Add JavaScript interactivity
* Apply basic SEO best practices
* Deploy using GitHub Pages
